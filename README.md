# EEG Epilepsy Detection using Machine Learning

This project analyzes EEG signals to detect epilepsy using signal processing and machine learning techniques.

## Features
- EEG preprocessing with MNE
- Spectral feature extraction (84 features)
- Random Forest classifier
- 98.75% accuracy
- Confusion matrix & evaluation

## Dataset
EEG Guinea-Bissau dataset (Zenodo)

## Tech Stack
Python, MNE, NumPy, Pandas, Scikit-learn, Matplotlib

## Results
- Accuracy: 98.75%
- Epilepsy detection model
- Healthcare analytics insights
